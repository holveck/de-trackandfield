# de-trackandfield

A small app for querying Delaware HS Track & Field champions (girls/boys) by event, meet, and year,
powered by a normalized dataset parsed from the "Delaware Track and Field Supersheet (6).xlsx" workbook.

## Run locally
pip install -r requirements.txt
streamlit run streamlit_app.py

## Deploy
On https://share.streamlit.io/:
- New app → Select this repo → main branch → `streamlit_app.py`
